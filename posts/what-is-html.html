What is HTML?

HTML is stands for Hyper Text Markup Language, it's a markup language used to creating Web pages. It defines basic structure of any website.

What ever you see on the websites it has been defined in HTML. It acts as the skeleton of a Web page. The images, text, and other content is "markup" in HTML. Learning HTML is the first step towards web designing.

Now, you have known the HTML is as skeleton of a Web page, so the website only HTML that looks very ugly. Other technology besides HTML are generally used to describe a web page's appearance/presentation that is CSS. If HTML is skeleton then CSS is the clothes of that skeleton. CSS is responsible for the design and colors and shape of a Web page.
